# AshenBlades - Konzept

## Intro
- Webapp-basiertes Browserspiel mit einer fortlaufend staerker werdenden Spielerfigur.
- Loot wird erbeutet, ausgeruestet und steigert direkt die Kampfkraft.
- Ausgeloggte Spieler laufen im Idle-Modus weiter und sammeln bis zum naechsten Level.
- Eingeloggte Spieler spielen aktiv Kampagne oder aktuelle Events.
- Build-Staerke entsteht aus Ausruestungsslots und den Werten Leben, Ruestung, Angriffskraft und Ausdauer.

## Spielablauf
- Wenn ein User nicht eingeloggt ist, laeuft die Spielerfigur im Idle-Modus weiter.
- Im Idle-Modus sammelt die Figur Loot bis zum naechsten Level.
- Wenn ein User eingeloggt ist, kann er aktiv die Kampagne spielen oder aktuelle Events absolvieren.

## Ausruestungsplaetze
Die Spielerfigur hat folgende Slots:
- Kopf
- Hals
- Cape
- Brust
- Waffen (ein- oder zweihaendiges Setup)
- 3 Traenke-Slots
- Fuesse

## Statuswerte
Der Spieler hat folgende Statuswerte:
- Leben
- Ruestung
- Angriffskraft
- Ausdauer
