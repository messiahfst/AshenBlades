# AshenBlades - Konzept

## Intro
- Webapp-basiertes Browserspiel mit einer fortlaufend staerker werdenden Spielerfigur.
- Loot wird erbeutet, ausgeruestet und steigert direkt die Kampfkraft.
- Ausgeloggte Spieler laufen im Idle-Modus weiter und sammeln bis zum naechsten Level.
- Eingeloggte Spieler spielen aktiv Kampagne oder aktuelle Events.
- Build-Staerke entsteht aus Ausruestungsslots und den Werten Leben, Ruestung, Angriffskraft und Ausdauer.

## Struktur
- Spielablauf: [game/GAME.md](game/GAME.md)
- Spielerfigur, Slots und Werte: [player/PLAYER.md](player/PLAYER.md)
